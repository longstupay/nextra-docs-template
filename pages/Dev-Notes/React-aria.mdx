# React-Aria

React-Aria是一个React Hooks库，它为你的设计系统提供了可访问的UI原语。它为许多常见的UI组件提供了可访问性和行为，让你可以专注于你独特的设计和样式。它实现了自适应的交互，确保为所有用户提供最佳的体验，包括支持鼠标、触摸、键盘和屏幕阅读器。

## 为什么使用React-Aria

使用React-Aria的原因是它可以帮助你构建符合标准、易于使用、兼容多种设备和平台的UI组件，而无需自己处理复杂的可访问性逻辑。React-Aria根据WAI-ARIA编写实践提供了可访问性和行为，包括完整的屏幕阅读器和键盘导航支持。所有组件都经过了各种屏幕阅读器和设备的测试，以确保为所有用户提供最佳的体验。

## 如何使用React-Aria

要使用React-Aria，你需要安装它作为一个依赖：

```bash
npm install @react-aria/hooks
```
然后你可以在你的代码中导入需要的Hooks，并将它们应用到你自己定义或引入的UI元素上。例如，如果你想创建一个按钮组件，你可以这样做：
```js
import {useButton} from '@react-aria/button';

function Button(props) {
  let {buttonProps} = useButton(props);

  return (
    <button {...buttonProps}>
      {props.children}
    </button>
  );
}
```
这样就创建了一个基本功能正常且可访问性良好按钮组件。

React-Aria组件示例
下面是一些使用React-Aria创建或改进过后常见UI组件示例：

开关
开关是一种常用来表示二元状态（开/关）选项或设置项。要创建一个开关组件，我们可以使用useSwitch Hook，并将其应用到一个input元素上：
```js
import {useSwitch} from '@react-aria/switch';

function Switch(props) {
  let {inputProps} = useSwitch(props);

  return (
    <label>
      <input {...inputProps} type="checkbox" />
      {props.children}
    </label>
  );
}
```
这样就创建了一个基本功能正常且可访问性良好开关组件。

### 复选框
复选框是一种常用来表示多选项（勾选/不勾选）选择或设置项。要创建一个复选框组件，我们可以使用useCheckbox Hook，并将其应用到一个input元素上：

```js
import {useCheckbox} from '@react-aria/checkbox';

function Checkbox(props) {
  let {inputProps} = useCheckbox(props);

  return (
    <label>
      <input {...inputProps} type="checkbox" />
      {props.children}
    </label>
  );
}
```
这样就创建了一个基本功能正常且可访问性良好复选框组件。它支持键盘和屏幕阅读器操作，并且可以自动处理一些常见的属性，如aria-checked和aria-invalid。
